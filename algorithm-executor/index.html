Executar projeto Java com JAR no CheerpJ

Esta versão é mais confiável do que compilar Java no navegador. Coloque seu app.jar na pasta app ao lado deste HTML e clique no botão.

Aguardando execução.

Como usar

  • 1. Gere o JAR do seu projeto Java no computador.
  • 2. Salve este arquivo como index.html.
  • 3. Crie a pasta app e coloque nela o arquivo app.jar.
  • 4. Rode com um servidor local, não abrindo o HTML direto no navegador.
estrutura esperada:
/
  index.html
  /app
    app.jar

Comandos para gerar o JAR

Se sua classe principal for Main:

javac *.java
jar cfe app.jar Main *.class

Se sua classe principal for Principal:

javac *.java
jar cfe app.jar Principal *.class

Depois, mova o app.jar para a pasta app.

Servidor local

python -m http.server 8080

Depois abra http://localhost:8080.
